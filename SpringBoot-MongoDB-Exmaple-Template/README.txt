
**Docker-Compose ausführen:**

1. Öffne PowerShell
2. Navigiere in powershell in den Ordner, in dem sich die "docker-compose.yml" Datei befindet
3. Führe folgenden Befefehl in PowerShell aus: docker-compose up
3.1 Um docker-compose im Hintergrund laufen zu lassen, kannst du folgenen Befehl nutzen: docker-compose up -d
4. Öffne in Powershell ein neues Fenster
5. Überprüfe mit dem Kommando "docker ps", ob die Docker Container für "mongoDB" und "mongo express" laufen
